##### Hi friends, this is my message after having played with this data for ~9 months

1. download all of the MODTRAN data you wish to work with, you should keep it in the data folder, but also move it to it's own folder titled with what version of the data you are looking at is
2. once you have this, you can run the `table_output` and `table_summary` functions under `modtran_analysis/tools/utils`. I also have a .py file `tabular_outputs` which will require you to provide how many scenes there are off the data, and create its own folders under the data file which you will need to rename as well 
3. you are free to look at the summaries of this data, and see what is going correctly and incorrectly in these files

I hope this code becomes helpful in the future
